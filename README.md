# Shopping Cart (new)

### Installing
* Install node packages  
`npm install`
* Create .env files (dev.env, test.env)  
* Run backend server  
`npm run dev`

### Testing
`npm test`

### API docs
[Project docs](http://shopping-cart-shopping-cart.7e14.starter-us-west-2.openshiftapps.com/docs/)

`npm run docs`
